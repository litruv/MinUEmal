# MinUEmal
A minimal, high performance starter template, with all the usual stuff turned off

- As Minimal Plugins as needed for games to run + FAB

## Disabled
- Lumen
- Nanite
- Motion Blur
- Auto Exposure
- Substrate
- SM6
- DX12
- Temporal Shading
- Ambient Occlusion
- Virtual Textures
- Sky Atmosphere
- Point Light Whole Scene Shadows
- Mesh Distance Fields
- Custom Depth
- Many Plugins
