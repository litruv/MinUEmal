# MinUEmal

A minimal, high-performance Unreal Engine starter template with all the usual expensive stuff turned off.

Built for fast iteration, fast editor startup, reduced shader compilation, and lower baseline overhead.

- Minimal plugins required for games to run (+ FAB)

## Disabled Features

- Lumen
- Nanite
- Motion Blur
- Auto Exposure
- Substrate
- Shader Model 6 (SM6)
- DirectX 12
- Temporal Super Resolution (TSR)
- Ambient Occlusion
- Virtual Textures
- Sky Atmosphere
- Point Light Whole Scene Shadows
- Mesh Distance Fields
- Custom Depth
- Many default plugins
